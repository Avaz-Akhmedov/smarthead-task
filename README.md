Описание проекта

Здравствуйте! Я выполнил задание. К сожалению, фронтенд не был реализован, 
чтобы сэкономить время, но все необходимые требования выполнены для демонстрации моих навыков.


Чтобы запустить проект, выполните следующие команды:

# Установить зависимости
composer install

# Создать файл окружения
cp .env.example .env

# Сгенерировать ключ приложения
php artisan key:generate

# Выполнить миграции и посеять данные
php artisan migrate --seed

# Запустить сервер разработки
php artisan serve

Команды Artisan

Я добавил команду для генерации нового токена Sanctum, чтобы легко создавать пользователей:

php artisan generate:token

Пользователи в базе данных после запуска сидеров

После выполнения сидеров в базе данных будут следующие пользователи:

ID	Роль	Примечание
1	Admin	Пользователь с правами администратора
2	Manager	Пользователь с ролью менеджера
3	—	    Пользователь без роли
